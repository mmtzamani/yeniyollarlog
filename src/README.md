# Yeni Yollar Logistics - Multilingual Corporate Website

A modern, professional, and fully responsive corporate website for Yeni Yollar Logistics, an international shipping and foreign trade company.

## Features

### Multilingual Support (3 Languages)
- **Turkish (TR)** - Default language
- **English (EN)** - International language
- **Persian (FA)** - RTL support included

### Pages
1. **Home** - Hero section, stats, services overview, why choose us
2. **About** - Company story, mission, vision, values
3. **Services** - Detailed service offerings with imagery
4. **Team** - Management team profiles
5. **Contact** - Contact form, info, map integration
6. **Tracking** - Shipment tracking interface
7. **Gallery** - Logistics operations imagery

### Services Offered
- Air Freight
- Ocean Freight
- Road Transport
- Customs Clearance
- Warehousing & Distribution
- Global Trade Consulting

### Design Features
- Premium, modern UI/UX design
- Brand colors: #1B4C7C (Dark Blue), #4CAF50 (Green), #889DB0 (Blue-Grey)
- Smooth animations with Motion/React
- Responsive design (mobile, tablet, desktop)
- WhatsApp floating action button
- Sticky header with language switcher
- RTL layout support for Persian
- Custom scrollbar styling
- Professional typography
- Shadow and hover effects

### Technical Stack
- React 18
- TypeScript
- React Router DOM
- Tailwind CSS v4
- Motion/React (animations)
- Lucide React (icons)
- Shadcn/ui components
- Sonner (toast notifications)

## Usage

The website automatically detects and applies the selected language. Use the language switcher in the header to change between Turkish, English, and Persian.

For Persian language, the entire layout automatically switches to RTL (right-to-left) mode.
